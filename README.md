Project Zomboid Challenge Generator

A browser-based mini-game for multiplayer Project Zomboid runs.
Assign randomized, roleplay driven challenges to each player. Sabotage, survival, manipulation, and mystery all designed to enhance immersion and chaos during roadtrip-style sessions.

Features
• 	Randomized challenge assignment based on in-game triggers (, , )
• 	Checkbox tracking for completed challenges
• 	Live scoring system based on difficulty
• 	Session persistence using  (no rerolling on refresh)
• 	Reset button for fresh runs
• 	Clean, responsive UI with zombie-themed styling

Designed for
• 	Multiplayer roleplay sessions
• 	Social deduction and hidden traitor mechanics
• 	Challenge-based storytelling
• 	Chaos-loving survivors

How to use
1. 	Open the site
2. 	Click “Get My Challenges”
3. 	Complete your 3 assigned tasks during the run
4. 	Track progress and score
5. 	Discuss scores at the end of the session
6. 	Reset when ready for a new round

Tech stack
• 	HTML, CSS, JavaScript
• 	GitHub Pages for hosting
• 	No backend required
